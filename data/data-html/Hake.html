Hake {FSAdata}R Documentation

Stock and recruitment data for Hake, 1982-1996.

Description

Stock and recruitment data for Hake (Merluccius merluccius), 1982-1996.

Format

A data frame with 15 observations on the following 3 variables.

  • year: a numeric vector of years 1982-1996
  • recruits: a numeric vector of the number of recruits in millions

  • spawn.biomass: a numeric vector of spawning biomass in thousand tonnes

Topic(s)

  • Stock-Recruit
  • Recruitment

Source

Cadima, E. 2003. Fish Stock Assessment Manual, FAO Fisheries Department. 131 pp.

Examples

data(Hake)
str(Hake)
head(Hake)
op <- par(mfrow=c(1,2),pch=19)
plot(recruits~year,data=Hake,type="l")
plot(recruits~spawn.biomass,data=Hake)
par(op)

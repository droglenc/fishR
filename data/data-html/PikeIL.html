PikeIL {FSAdata}R Documentation

Catch and effort data for Northern Pike from Island Lake, NB.

Description

Catch and effort data for nine removal events of Northern Pike (Esox lucius) from Island Lake, NB.

Format

A data frame with 9 observations on the following 3 variables:

  • date: Capture date.

  • nets: Number of fyke nets fished on that date.

  • catch: Number of captured and removed pike.

Topic(s)

  • Depletion methods
  • Leslie method
  • DeLury method
  • Population size
  • Abundance
  • Catchability

Source

From Table 1 of Peterson, J., M. Taylor, and A. Hanson. 1980. Leslie population estimate for a large lake. Transactions of the American Fisheries Society, 109:329-331.

Examples

data(PikeIL)
str(PikeIL)
head(PikeIL)
